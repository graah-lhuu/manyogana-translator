# 万葉仮名変換ツール

将现代日语假名转换为万叶假名（上代特殊假名）。

## 功能
- 平假名 → 万叶假名
- 片假名 → 万叶假名  
- 汉字保持原样
- 支持拗音、促音、拨音
- 简单易用的GUI界面

## 使用方法
```bash
pip install -r requirements.txt
python main.py
```
